# OrthoLife (OrthoWear)

OrthoLife is a comprehensive Orthodox Christian companion application specifically designed for Wear OS devices. It brings essential spiritual resources, liturgical information, and prayer tools directly to your wrist.

## Features

- **☦️ Orthodox Calendar:** Support for both **Julian** (Old Calendar) and **Revised Julian** (New Calendar).
- **📖 Daily Readings:** Stay connected with the daily liturgical cycle through appointed Gospel and Epistle readings.
- **🙏 Prayer Rope (Komboskini):** A digital prayer rope interface to assist in your daily rule of prayer.
- **📜 Holy Bible:** Accessible scripture readings optimized for wearable screens.
- **⛪ Prayers & Icons:** A collection of traditional Orthodox prayers and sacred icons.
- **😇 Lives of Saints:** Daily commemorations and hagiographies of the saints.
- **⌚ Watch Face Complications:** Display the current feast or liturgical day directly on your favorite watch face.
- **⚙️ Customizable Experience:** Adjustable text sizes and multi-language support (including Korean and English).

## Tech Stack

- **Platform:** Wear OS (Min SDK 31)
- **Language:** Kotlin & Java
- **UI:** [Jetpack Compose for Wear OS](https://developer.android.com/training/wearables/compose)
- **Networking:** Retrofit & Gson
- **Architecture:** MVVM / Modern Android Development practices

## Getting Started

### Prerequisites

- Android Studio Ladybug or newer
- Android SDK 31+
- A Wear OS device or emulator

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Wadotu/OrthoWear.git
   ```
2. Open the project in Android Studio.
3. Sync Project with Gradle Files.
4. Run the `app` module on your Wear OS device or emulator.

## Project Structure

- `presentation/`: Contains UI logic and Compose screens.
- `res/raw/`: Spiritual texts and data files (Bible, Prayers).
- `res/values/`: String resources and theme configurations.
